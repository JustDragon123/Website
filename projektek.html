Arduino "Napraforgó"

 Manapság egyre nagyobb jelentőséget kap a megújuló energia és annak hatékony előállítása. A projektem egy napelem szabályzó rendszert modellez, amely egy tengely mentén automatikusan a fényforrás irányába fordul. Az eredeti célom egy saját, otthoni tó levegőztető rendszerének optimalizálása volt, azáltal, hogy a napelemes tápegység folyamatosan a nap felé forduljon. Ezzel biztosítható, hogy a levegőztető rendszer maximális teljesítménnyel működjön anélkül, hogy emberi beavatkozásra lenne szükség.

Működési elve

 Két LDR diódát helyezünk a napelem két szélére, párhuzamosan annak síkjával. (Mivel a modellemen nincs felszerelve napelem, a diódákat egy kis méretű dugaszolópanel két szélére rögzítettem.) A két LDR dióda fényérzékelési értékeit figyelve jelet küldünk a szervómotornak, amely az érzékelt fényintenzitás különbsége alapján elmozdul abba az irányba, ahol az LDR több fényt kap.

Amit használtam hozzá:

  • 1db Arduino (Én Arduino Mega 2560-at használtam)
  • 2db LDR Fotoellenállás
  • 2db 10kΩ ellenállás
  • 1db 5V-os szervó
  • 1db Kisméretű dugaszolópanel
  • 11db jumper vezeték

Kapcsolási rajz

A kód


#include <Servo.h>  //Beimportáljuk a szervó működéséhez szükséges könyvtárat
Servo szervomotor ;         //A szervót inicialízáljuk "szervomotor" néven
int keletLDRPort = 5;
int nyugatLDRPort = 1;
int kelet = 0;
int nyugat = 0;
int kulombseg = 0;
int minpos = 14;
int maxpos = 180;  
int pos = 120; //120 egyenes felfele
int i;

void setup()
{
  szervomotor.attach(2);
  szervomotor.write(pos); //A szervó első lefutásakor beáll egyenesen felfele
  Serial.begin(9600);
}

void loop()
{
  kelet = analogRead(keletLDRPort);  //beolvassuk a "keleti" felén lévő LDR-t
  nyugat = analogRead(nyugatLDRPort); //beolvassuk a "nyugati" felén lévő LDR-t
  kulombseg = (kelet - nyugat)/2;       //A külömbséget kiszámoltatjuk majd el osszuk kettővel a kisebb számmal való dolgozáshoz
  
    for (i = 0; i < 2; i+=1){
        if (kulombseg > 10){              //Itt ha a külömbség nagyobb mint 10 akkor elfordítja a szervót a servó elejétől nézve jobbra
        pos -= 2;
        if (pos > minpos){
            szervomotor.write(pos);
        } else {
          pos += 2;
        }
      } else if (kulombseg < -10){        //Itt ha a külömbség kissebb mint -10 akkor elfordítja a szervót a servó elejétől nézve balra
        pos += 2;                               //széljegyzet: Azért nem 0 mivel így nem mozog be a szervó minden kissebb árnyéknál
        if (pos < maxpos){
            szervomotor.write(pos);
        } else {
          pos -= 2;
        }
      }                             
    }
    
  Serial.print("Kelet: "); 
  Serial.println(kelet);         //visszajelzést ad az arduino a keleti oldalán lévő LDR értékéről
  Serial.print("Nyugat: ");
  Serial.println(nyugat);       //visszajelzést ad az arduino a nyugati oldalán lévő LDR értékéről
  Serial.print("Servó poziciója: ");
  Serial.println(pos);          //visszajelzést ad az arduino a szervó helyzetéről
  Serial.println(kulombseg);    //visszajelzést ad az arduino a két LDR közötti külömbslgről
  delay(100);
  
}
